
Smart Network Traffic Analyzer

About:-
This is a beginner-friendly Python project that analyzes network traffic data from CSV files and identifies potentially suspicious records based on packet size.

Features:-
- Load CSV data
- Remove duplicate records
- Handle missing values
- Calculate basic statistics
- Detect high-risk packets
- Save suspicious records in SQLite
- Generate graphs using Matplotlib
- Export final report to CSV
 
Technologies Used:-
- Python
- Pandas
- NumPy
- Matplotlib
- SQLite
- 
Learning Outcome:-
This project helped me practice data analysis, database integration, visualization, and basic anomaly detection using Python.
